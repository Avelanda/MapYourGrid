### Quick contributions to the repository
> If you like the project and don't have the time or lack certain skills to contribute to the website, there are still many ways to contribute, which we would also be very happy about:
> - Star the project
> - Talk about it on social media
> - Create an "Issue" to add some comments or suggestions

### Contributions to MapYourGrid
> - Map the power grid!
> - Find more "impact studies". If you know of an organisation/company that uses OpenStreetMap grid data, then please mention it as an issue.
> - Improve the quality health indicator by adding/finding more recent numbers for the line length of a country's high-voltage grid. [Sheet](https://docs.google.com/spreadsheets/d/1qmVIQ2_ynVVfbTWcMXJQWb4Sq0Dq-1fu8zgZ9J_0cZI/edit?gid=0#gid=0)
> - Add ["Good first Lines"](https://mapyourgrid.org/good-first-lines/#want-to-add-a-good-first-line) for beginners or anyone to map
> - Find grid maps for countries, and add/create an issue for the [Awesome list](https://github.com/open-energy-transition/Awesome-Electrical-Grid-Mapping)
> - Find ways to make the Map It page faster (lighter geojsons, topojson perhaps etc..)
> - I'm sure you can find other creative ways!
